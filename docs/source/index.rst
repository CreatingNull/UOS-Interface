UOS Interface Documentation
===========================

Python Interface for remote communication and control of a `UOS Compliant Microcontroller <https://wiki.nulltek.xyz/projects/uos/>`_.

The uosinterface tooling source code is divided into two packages:

*	:doc:`hardware/index` - Containing device definitions and low level abstraction which can be imported to introduce UOS automation directly on a client machine.

*	:doc:`webapp/index` - Containing a high level web server which can be used to provide network access to a uos controller via a dashboard or API requests.
